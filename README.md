spaceship
=========

Very simple arcade game built with Gosu (Ruby).

Built with
----------

* Ruby
* [gosu](http://www.libgosu.org/)
* [hasu](https://github.com/michaelfairley/hasu)


How to run this game
--------------------

```
git clone https://github.com/mrhead/spaceship
cd spaceship
bundle install
ruby game.rb
```
